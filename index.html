About

Web portfolio ini saya buat sebagai media belajar saya tentang web developer khususnya untuk front end nya. Tentunya masih banyak yang harus saya perbaiki dari web ini. Terutama pada bagian isi konten nya dan juga beberapa bagian yang belum bisa cukup responsive. Harapannya kedepannya dengan semakin sering adanya perbaikan menjadikan web ini menjadi lebih baik dan terus lebih baik kedepannya. Tidak ingin yang terlalu muluk-muluk, hanya ingin lebih memahami setiap bagian dari proses pembuatan web ini. Sehingga, kedepannya jika ada yang membutuhkan bantuan untuk mendevelope sebuah web dapat saya bantu.

Saat ini masih belum ingin untuk membuka jasa design web nya ataupun mendevelope web nya. Soalnya saya juga masih dalam tahap belajar. Harapannya dapat sesegera mungkin memahami setiap bagian yang dibutuhkan untuk menyelesaikan sebuah permasalahan web secara sederhana.
